  Step 5 — find_best_trust_signal() (matcher.py:381)
                                                                                               
  For Spice Garden — Layer 1 direct_trust hit:
                                                                                               
  -- _DIRECT_TRUST_SQL                                      
  SELECT te.to_user_id, u.name, te.weight, i.outcome, i.visited_at, i.intent_parsed            
  FROM trust_edges te                                                                          
  JOIN users u ON u.id = te.to_user_id
  JOIN interactions i ON i.user_id = te.to_user_id AND i.restaurant_id = 'spice-garden-id'     
  WHERE te.from_user_id = 'arjun-id'                                                           
    AND te.status = 'active' AND te.domain = 'restaurants'
    AND i.outcome IN ('positive', 'neutral')                                                   
  ORDER BY te.weight DESC LIMIT 1                           
                                                                                               
  Result: Priya visited Spice Garden for a business dinner, outcome=positive                   
                                                                                               
  TrustSignal(                                                                                 
      signal_layer      = 'direct_trust',                   
      trusted_user_name = 'Priya',                                                             
      edge_weight       = 0.82,          # Arjun→Priya trust edge weight
      hops              = 1,                                                                   
      outcome           = 'positive',                       
      visited_at        = '2026-03-15',                                                        
      interaction_intent = {             # what Priya searched for when she went
          "fields": {                                                                          
              "cuisine":     {"value": ["North Indian"], "constraint": "hard"},
              "occasion":    {"value": ["business"],     "constraint": "soft"},                
              "noise_level": {"value": "quiet",          "constraint": "soft"},                
          }                                                                                    
      }                                                                                        
  )                                                         

  Layer 1 hit → skip Layers 2-5 entirely.                                                      
  
  For Punjabi Tadka — Layer 1 miss, Layer 2 network_trust hit:                                 
                                                            
  No one in Arjun's direct trust network visited Punjabi Tadka. Recursive CTE fires:           
                                                            
  trust_paths traversal:                                                                       
    Arjun → Priya  (weight=0.82, hops=1)                    
    Arjun → Kiran  (weight=0.79, hops=1)                                                       
    Priya → Rahul  (weight=0.75, hops=2)  ← Rahul visited Punjabi Tadka!                       
                                                                                               
  path_score = avg_weight × 0.85^(hops-1)                                                      
             = ((0.82+0.75)/2) × 0.85^1                                                        
             = 0.785 × 0.85 = 0.667                                                            
                                                            
  TrustSignal(                                                                                 
      signal_layer      = 'network_trust',                  
      trusted_user_name = 'Rahul',                                                             
      edge_weight       = 0.667,         # path_score
      hops              = 2,                                                                   
      outcome           = 'positive',                       
      visited_at        = '2026-02-20',                                                        
      path_count        = 1,
      interaction_intent = {                                                                   
          "fields": {                                       
              "cuisine":  {"value": ["North Indian"], "constraint": "hard"},
              "occasion": {"value": ["casual"],       "constraint": "soft"},                   
          }
      }

  Step 6 — compute_intent_similarity() (matcher.py:475)                                        
                                                            
  Compares Arjun's current intent vs what the trusted person searched for.
                                                                                               
  Spice Garden (Priya's past intent vs Arjun's current intent):                                
                                                                                               
  Priya searched: occasion=business, noise_level=quiet, cuisine=North Indian                   
  Arjun wants:    occasion=business, noise_level=quiet, cuisine=North Indian                   
  
  cuisine:     1.0 × 0.20 = 0.20                                                               
  area:        1.0 × 0.10 = 0.10                            
  occasion:    business vs business → sim=1.0 × 0.35 = 0.35                                    
  noise_level: quiet vs quiet → 1.0 × 0.05 = 0.05                                              
                                                                                               
  similarity = 0.70/0.70 = 1.00   ← perfect context match                                      
                                                                                               
  Punjabi Tadka (Rahul's past intent vs Arjun's current intent):                               
                                                            
  Rahul searched: occasion=casual, cuisine=North Indian                                        
  Arjun wants:    occasion=business, noise_level=quiet      

  cuisine:     1.0 × 0.20 = 0.20                                                               
  area:        1.0 × 0.10 = 0.10
  occasion:    business vs casual → sim_map=0.4 × 0.35 = 0.14                                  
  noise_level: Rahul didn't specify → neutral score 0.5 × 0.05 = 0.025                         
                                                                                               
  similarity = 0.465/0.70 = 0.664   ← Rahul went casually, Arjun wants business                
                                                                                               
  This is the key insight of intent-contextualized trust. Rahul's recommendation is discounted 
  because the context of his visit doesn't match Arjun's need.

  Step 7 — score_trust_signal() (matcher.py:532)
                                                
  Spice Garden (direct_trust, cap=0.95, similarity=1.00):
                                                                                               
  edge_component    = 0.82 × 0.50  = 0.410
  outcome_component = 0.80 × 0.30  = 0.240   (positive outcome)                                
  recency_days      = 47 days ago                                                              
  recency_component = (1 - 47/180) × 0.20 = 0.739 × 0.20 = 0.148                               
                                                                                               
  raw = 0.410 + 0.240 + 0.148 = 0.798                                                          
  raw × similarity = 0.798 × 1.00 = 0.798                                                      
  trust_score = min(0.95, 0.798) = 0.798                                                       
                                                            
  Punjabi Tadka (network_trust, cap=0.75, similarity=0.664):                                   
  
  edge_component    = 0.667 × 0.50 = 0.334                                                     
  outcome_component = 0.80  × 0.30 = 0.240                  
  recency_days      = 70 days ago                                                              
  recency_component = (1 - 70/180) × 0.20 = 0.611 × 0.20 = 0.122
                                                                                               
  raw = 0.334 + 0.240 + 0.122 = 0.696                       
  raw × similarity = 0.696 × 0.664 = 0.462   ← discounted for context mismatch                 
  trust_score = min(0.75, 0.462) = 0.462
